# Xcode 11 shortcuts

## Legend
⌘ = Command
⌥ = Option/Alt
⌃ = Control
⇧ = Shift
↩ = Enter
←↑↓→ = Arrow keys


## Editor (Code Editing)
|Action|Shortcurt|
|-|-:|
| Code completion | ⌃ + Space OR Esc |
| Move line down | ⌥ + ⌘ + ] |	
| Move line up | ⌥ + ⌘ + [ |
| Fold code block | ⌥ + ⌘ + ← |  
| Unfold code block | ⌥ + ⌘ + → |
| Comment line/selection | ⌘ + / |	
| Balance indent | ⌃ + I |	

## Editor (View Layout)
|Action|Shortcurt|
|-|-:|
| Show Editor Only | ⌘ + ↩︎ |
| Toggle Canvas/Preview | ⌥ + ⌘ + ↩︎ |
| Toggle Assistant | ⌃ + ⌥ + ⌘ + ↩︎ |
| Toggle Minimap | ⇧ + ⌃ + ⌘ + M |
| Toggle Authors | ⇧ + ⌃ + ⌘ + A |
