# Xcode 11 shortcuts

## Legend
⌘ = Command
⌥ = Option/Alt
⌃ = Control
⇧ = Shift
↩ = Enter
←↑↓→ = Arrow keys


## Editor (Code Editing)
|Action|Shortcurt|
|-|-:|
|	Code completion | ⌃ + Space |
|	Move line down | ⌥ + ⌘ + ] |	
|	Move line up | ⌥ + ⌘ + [ |
|	Comment line/selection | ⌘ + / |	
|	Balance indent 	| ⌃ + I |	

## Editor (Window Layout)
|Action|Shortcurt|

| Toggle Minimap | ⌃ + ⇧ + ⌘ + M |
